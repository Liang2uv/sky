# sky

使用Auto.js提供的无障碍权限实现在Sky光遇中自动弹奏[SkyStudio](https://play.google.com/store/apps/details?id=com.Maple.SkyStudio) 导出的曲谱

## 使用

1. 点击`Autojs.apk`并安装

2. 打开Auto.js侧边栏，为Auto.js开启**无障碍服务**和**悬浮窗权限**。

3. 在手机根目录新建文件夹`skyMusic`，将`sheets`文件夹里的所有`.txt`文件复制到`skyMusic`里

4. 打开Auto.js，点击右下角的导入，依次导入文件`sky.js`和`sky-clear.js`

5. 打开`光遇`，调出弹琴界面

6. 在Auto.js悬浮窗中运行脚本`sky.js`

7. 演奏中途请勿操作屏幕，如果需要中断，请点击Auto.js悬浮窗的 **X** 按钮

## 清除缓存

本脚本将记录第一次按键坐标的设置信息，如果需要重新设置，请运行`sky-clear.js`文件

## 关于乐谱

乐谱文件必须是未加密的`.txt`文件（PS：烛子老师的乐谱大部分都是加密的），把乐谱文件复制到`skyMusic`文件夹下就欧了
